   - Miner profitability?
   - Mining difficulty?
   - Fee pressure?

2. **If block subsidy halves (2028), what happens to:**
   - Miner revenue?
   - Security budget?
   - Need for higher fees?

3. **If transaction fees increase 5x, what happens to:**
   - Lightning Network adoption?
   - On-chain activity?
   - User behavior?

**If you can reason through these ‚Üí You understand Bitcoin economics!**
