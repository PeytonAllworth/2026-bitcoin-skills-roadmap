### Step 1: Set Up Your Environment (1 hour)

**Install Python & Jupyter:**

1. **Download Python:**
   - Go to https://www.python.org/downloads/
   - Download Python 3.10 or newer
   - Install (check "Add Python to PATH" on Windows)

2. **Install Jupyter:**
   ```bash
   pip install jupyter notebook
   pip install pandas numpy matplotlib seaborn
   ```

3. **Test it works:**
   ```bash
   jupyter notebook
   ```
   - Should open in your browser
   - If it works, you're good!

4. **Install a code editor (optional but recommended):**
   - VS Code: https://code.visualstudio.com/
   - Install Python extension in VS Code

**Alternative (easier):**
- Install Anaconda: https://www.anaconda.com/download
- Comes with everything pre-installed
- Recommended if you're new to Python
